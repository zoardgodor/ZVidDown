Video Downloader – egyszerű, gyors és hatékony

Mi az a ZVidDown?

A ZVidDown egy egyszerű, grafikus felületű videóletöltő alkalmazás, amellyel YouTube-ról és más támogatott weboldalakról tölthetsz le videót, hangot vagy csak videót.

Fő funkciók

  • Videó + hang letöltése
  • Csak hang (MP3) letöltése
  • Csak videó letöltése
  • Felbontás kiválasztása (ha elérhető)
  • Kimeneti mappa megadása
  • Letöltési folyamat kijelzése
  • Széles körű oldal támogatás (YouTube, Videa, stb.)

Technológiák

  • Python 3
  • Tkinter (GUI)
  • yt-dlp
  • ffmpeg, ffprobe

What is ZVidDown?

ZVidDown is a simple graphical video downloader application that allows you to download video, audio, or video-only content from YouTube and other supported websites.

Main Features

  • Download video + audio
  • Download audio only (MP3)
  • Download video only
  • Resolution selection (if available)
  • Choose output folder
  • Download progress display
  • Broad website support (YouTube, Videa, etc.)

Technologies

  • Python 3
  • Tkinter (GUI)
  • yt-dlp
  • ffmpeg, ffprobe
